
# Back-end

### Instruções para execução do projeto

  

#### Realize o clone do projeto
```bash
git clone https://github.com/magobr/back-end.git
```
#### Instale os pacotes necessários
```bash
npm install
```
ou
```bash
yarn install
```
#### Inicie o projeto NodeJs
```bash
node app
```
ou para nível de desenvolvimento
```bash
nodemon app
```
